안녕하세요, **김봉섭**입니다.
응용 소프트웨어 개발자로 안전보건 관리 시스템, 모바일 앱, 클라우드 전환 사업 등 다양한 프로젝트에 참여해 왔습니다.

## 이렇게 일합니다

AI 도구를 쓰는 데 그치지 않고, **생산성을 높이는 도구를 직접 설계하고 배포**합니다.

| 분야 | 내용 | 성과 |
| --- | --- | --- |
| 멀티 LLM 오케스트레이션 | Claude Code 안에서 다른 LLM을 호출하는 [oh-my-bridge](/posts/claude-code-multi-llm-orchestration) 플러그인 직접 개발. Claude(mechanics-driven: 체크리스트·절차 명시)로 계획·디버깅을 처리하고, 코드 생성 단계는 GPT(principle-driven: 목표 제시·방법 자율)로 분기 | Opus 단독 대비 토큰 비용 **약 50% 절감**, 코드 품질 향상 |
| 사내 디자인 시스템 구축 | 사내 디자인 시스템 DDS(Duego Design System) 도입을 직접 제안·설계하고, 팀원 배포용 Claude Code 플러그인으로 패키징 | 반복 UI 개발 시간 **약 60% 단축**, UI 일관성 향상 |
